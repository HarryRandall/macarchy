# Turbonite
Turbonite is a color developed by Porsche Design, created to represent a modern, high-performance aesthetic with a subtle, metallic tone that reflects both precision and innovation.

# Installation

To install this theme, simply use the omarchy-theme-install command:

```bash
omarchy-theme-install https://github.com/HANCORE-linux/omarchy-turbonite-theme.git
```
